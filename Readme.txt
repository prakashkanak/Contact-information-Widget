== Description ==

**Contact Info**

This plugin shows your contact information with icons. Add the widget to a widget position like the footer or sidebar, fill in the info and click save. Your contact information with icons is displayed on the front-end of your website!

**Features**

This plugin allows the user to add:

* Title
* Name
* Email
* Phone
* Fax
* Address


== Installation ==

Upload the .zip file with a FTP program in the folder wp-content/plugins or upload via Worpress backend.

== Frequently Asked Questions ==

**The plugin is not working on my website how come?**

1. Check if it's enabled.
2. Check if you run the latest version of Wordpress
